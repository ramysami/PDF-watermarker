# PDF watermarker
This python script allows you to apply multiple watermarks to multiple source PDF files (input) and have the output nicely organized in "output" folder and named correctly.
# How to use it 
1. Add the source files in "input" folder
2. Put the PDFs for the watermarks in a folder called "watermarks"
3. Open "PDF watermarker"

Finished files are added to a folder named "output"
